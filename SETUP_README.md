# LostCity Launcher — Setup Guide

## What's Included

This release contains:
- **engine/** — The LostCity game server (TypeScript, Bun runtime)
- **content/** — Game data (models, maps, scripts, audio, configs)
- **client/** — The Java game client (Gradle project)
- **dashboard/** — The Launcher UI source (Next.js)
- **electron/** — Standalone desktop wrapper

## What's NOT Included

- **OSRS Cache** (~500MB) — Downloaded on demand via the Launcher's GitHub API
- **node_modules/** — Installed automatically by `bun install`
- **Build outputs** — Generated by `bun run build`

## Quick Start

### Prerequisites
1. **Bun** (JavaScript runtime) — https://bun.sh
2. **Java JDK 11+** (for the client) — https://adoptium.net
3. **Gradle** (included in client/ via gradlew)

### Setup

1. **Install engine dependencies:**
   ```bash
   cd engine
   bun install
   ```

2. **Start the Launcher:**
   ```bash
   cd dashboard
   bun install
   bun run dev
   ```
   Open http://localhost:3000 in your browser.

3. **Enable CRC Bypass** (required for modding):
   - Go to the **Models** tab in the Launcher
   - Toggle **Bypass CRC Checks** to ON
   - This allows modified cache files to be accepted by the client

4. **Import OSRS Content** (optional):
   - In the **Models** tab, click **Download Models** for the Tormented Demon
   - Click **Apply Import** to register the NPC
   - The Launcher downloads only the needed files from GitHub

5. **Build the game:**
   - Go to the **Play** tab
   - Click **Build Game** (runs `bun start` which includes `bun install`)
   - Or click **Clean Build** for a fresh build (wipes cache + .file_store_32)

6. **Play:**
   - Click the large **Play** button (builds + launches the client)
   - Or run manually: `cd client && ./gradlew run`

## Standalone Electron App

To build a standalone .exe:

```bash
cd electron
npm install
npm run build-win    # Windows .exe (NSIS installer)
npm run build-linux  # Linux AppImage
```

The built app will be in `electron/dist/`.

## Troubleshooting

### "CRC check failed" error
Enable CRC Bypass in the Models tab.

### "Module not found" errors
Run `bun install` in the engine directory.

### Client won't launch
Ensure Java JDK 11+ is installed and JAVA_HOME is set.

### Map changes don't appear
Run **Clean Build** from the Play tab (wipes the cache and .file_store_32).
